# AI Gateway 大模型 Prompt 说明

## 云端服务

- 服务：火山引擎边缘大模型网关（AI Gateway）。
- 调用位置：成员 C 后端。
- 调用方式：对话模型使用 OpenAI 兼容的 Chat Completions 格式。
- 实际 Base URL、网关访问密钥和模型标识：以 AI Gateway 控制台“查看代码”为准。

## 输入数据

AI Gateway 调用的云端模型不直接接收单一传感器值，而是接收后端融合后的结构化摘要：

- 光照：平均/最低/最高照度、适宜比例。
- IMU：平均活动度、主要模式。
- 专注：累计专注分钟数、完成周期数。
- 电量：平均电量。

这与小组方案中的“光环境 + 姿态/活动 + 专注状态 + 电量”多源感知数据处理一致。

## 系统 Prompt

```text
你是 FocusCube 智能光环境与专注管理助手。
你会收到由光照、IMU 活动/姿态、专注计时和电量融合得到的结构化日统计。
请只基于输入数据生成简洁、可执行、不过度推断的中文复盘。
必须返回 JSON，格式为：
{
  "report_text": "80-160 字中文复盘",
  "suggestions": ["建议1", "建议2"]
}
不要输出 Markdown，不要虚构未提供的健康、身份或环境信息。
```

## 正式验收

正式比赛必须配置真实的 `FOCUSCUBE_LLM_BASE_URL`、`FOCUSCUBE_LLM_API_KEY` 和 `FOCUSCUBE_LLM_MODEL`，并展示至少一次 AI Gateway 成功调用。规则报告只用于无网络或接口异常时的演示兜底，不能代替云端大模型接入证明。
