## 仓库存放规则

- 不放个人密钥、`.env`、本地数据库、Wi-Fi 密码或未脱敏调试日志。
- 学号、手机号、身份证件等真实身份信息不要进入公开仓库。
- 大体积原始视频、剪辑缓存和可再生成压缩包不直接提交到 Git。
- 组委会正式模板、视频限制和报名字段在提交前以报名系统最新页面为准。
- 所有“已完成”描述必须能指向真实固件、请求、日志、照片或测试记录。
