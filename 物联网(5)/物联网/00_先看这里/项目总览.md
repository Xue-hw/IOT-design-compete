# 项目总览

## 一句话定位

FocusCube 是一个基于 ESP32-S3 EYE/Cube 与 P4 七寸屏的智能光环境与专注管理系统。EYE 用于开发测试，Cube 用于最终演示且是 EYE 的功能子集；当前只要求其中一个主设备在线。P4 负责正式屏幕展示，后端和云端大模型负责生成复盘与提醒。

## 当前系统主线

```text
当前选中的 EYE 或 Cube
  采集光照、姿态、触摸/按键、专注计时、电量
        ↓
后端服务
  接收 telemetry，聚合当天数据
        ↓
云端大模型
  生成“光环境 + 专注效率”复盘、建议和提醒
        ↓
Web 看板 + P4 七寸屏
  展示数据、曲线、复盘、提醒和系统状态
        ↓
S3/P4
  显示或响应提醒
```

## MVP 演示必须出现

- EYE 本地功能验收已通过；完成后端上传闭环后再统一下发正式绑定，Cube 作为最终演示主设备。
- S3 上传或导出结构化感知数据。
- 后端能收到数据并生成报告输入。
- 大模型能输出中文复盘和建议。
- P4 七寸屏能展示 S3 数据、大模型复盘或提醒状态。
- Web 看板能展示曲线、指标和大模型结果。
- 设备端能显示或响应提醒。
- 全流程录成 1-5 分钟视频。

## 暂缓功能

这些不作为 7 月 27 日前的主线：

- S3 大规模固件重构。
- 在 EYE 上传闭环和主设备 ID 规则固化前扩展正式设备绑定。
- 基于“所有 S3 都有屏幕”的假设设计功能；EYE 屏幕未启用，Cube 无屏幕。
- 完整移动端 APP。
- 所有音视频能力都接入大模型。
- 复杂本地 AI 模型训练。
