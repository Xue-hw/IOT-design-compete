# 项目总览

## 一句话定位

FocusCube 是一个基于 S3 立方体和 P4 七寸屏的智能光环境与专注管理系统。S3 负责稳定采集，P4 负责屏幕展示，后端和云端大模型负责生成复盘与提醒。

## 参赛赛道

- 比赛：2026 年全国大学生物联网设计竞赛
- 赛道：乐鑫科技赛道
- 关键词：AIoT、ESP32-S3、ESP32-P4、传感器融合、云端大模型、上下行交互

## 当前系统主线

```text
S3 立方体
  采集光照、姿态、触摸/按键、专注计时、电量
        ↓
后端服务
  接收 telemetry，聚合当天数据
        ↓
云端大模型
  生成“光环境 + 专注效率”复盘、建议和提醒
        ↓
Web 看板 + P4 七寸屏
  展示数据、曲线、复盘、提醒和系统状态
        ↓
S3/P4
  显示或响应提醒
```

## MVP 演示必须出现

- S3 立方体实物和已烧录功能，不对 S3 做大改。
- S3 上传或导出结构化感知数据。
- 后端能收到数据并生成报告输入。
- 大模型能输出中文复盘和建议。
- P4 七寸屏能展示 S3 数据、大模型复盘或提醒状态。
- Web 看板能展示曲线、指标和大模型结果。
- 设备端能显示或响应提醒。
- 全流程录成 1-5 分钟视频。

## 暂缓功能

这些不作为 7 月 27 日前的主线：

- S3 大规模固件重构。
- 重新设计 S3 外壳尺寸或屏幕尺寸。
- 完整移动端 APP。
- 所有音视频能力都接入大模型。
- 复杂本地 AI 模型训练。
