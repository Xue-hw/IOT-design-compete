# FocusCube P4 七寸屏工程启动说明

> 当前设备口径：P4 每次只展示 `CONFIG_FOCUSCUBE_DEVICE_ID` 选中的一个数据源。代理联调阶段继续使用 `focuscube-c3-proxy-01`；它只用于追踪 C3/AS7341 光照来源，不是第三个产品节点。EYE/Cube 正式绑定等 EYE 功能测试后统一下发。

## 工程位置

本仓库：`04_技术实现/P4七寸屏工程`

目标板卡：Waveshare ESP32-P4-WIFI6-Touch-LCD-7B  
ESP-IDF：v6.0.2  
后端：`http://82.156.238.244/focuscube`

本工程基于 Waveshare 官方
[`10_lvgl_demo_v9`](https://github.com/waveshareteam/ESP32-P4-WIFI6-Touch-LCD-7B/tree/main/examples/ESP-IDF/10_lvgl_demo_v9)
示例开发，仓库内保留 FocusCube 所需源代码、依赖锁定和 BSP 增量，不提交自动下载的
`managed_components` 或本机构建目录。

`components/espressif__esp_wifi_remote` 是 `esp_wifi_remote 0.14.5` 的最小本地覆盖，
仅用于修复 ESP-IDF v6 已移除 `esp_interface.h` 导致的编译失败；其上游许可证保留在组件目录中。

`components/espressif__esp_hosted` 是 `esp_hosted 1.4.7` 的最小本地覆盖，
用于补充 ESP-IDF v6 所需的 `esp_driver_sdmmc` 组件依赖。

`components/espressif__esp_lvgl_port` 是 `esp_lvgl_port 2.8.0` 的最小本地覆盖，
用于兼容 LVGL 9.2 的颜色格式枚举变更。

`components/chmorgan__esp-audio-player` 是 `esp-audio-player 1.0.7` 的最小本地覆盖，
用于补充 ESP-IDF v6 所需的 `esp_driver_i2s` 组件依赖。

## 第一版页面

- 状态区：S3 在线状态、实时光照、光照标签、专注状态、剩余时间和电量。
- AI 复盘区：当日 `report_text` 和第一条 `suggestions`。
- 提醒区：按 `priority` 从高到低显示提醒；内容超出面板时可竖向触摸滚动。

刷新周期：

- `/api/v1/status`：3 秒。
- `/api/v1/reminders`：10 秒。
- `/api/v1/report/daily`：启动时一次，之后每 60 秒。

请求失败时保留上次成功内容，并显示“网络连接异常，正在重试”；后端恢复后自动继续刷新。

## 配置 BUPT-portal 校园网

进入配置菜单：

```sh
source "$HOME/.espressif/v6.0.2/esp-idf/export.sh"
idf.py -B "$HOME/.esp-idf-builds/focuscube-p4" menuconfig
```

工程默认连接无线网 `BUPT-portal`，Wi-Fi 密码保持为空。在
`FocusCube P4 Configuration` 中只填写：

- `BUPT portal account`：校园网账号。
- `BUPT portal password`：校园网密码。

账号密码仅保存在本机 `sdkconfig` 和烧录后的设备中，不要发送给他人、截图或提交到公开仓库。
也可在终端执行本地配置脚本，密码输入过程不会显示：

```sh
cd "物联网(5)/物联网/04_技术实现/P4七寸屏工程"
./配置北邮校园网.sh
```

校园网认证流程为：连接开放 Wi-Fi、获取门户跳转和 Cookie、POST 账号密码、
再次访问探测页验证结果。认证中断时每 10 秒重试，正常时每 45 秒复查。

> 注：目前 BUPT 门户登录接口为明文 HTTP，仅应在可信的北邮校园网中使用。


## 烧录和查看日志

先用 `ls /dev/cu.*` 确认当前串口，再执行：

```sh
source "$HOME/.espressif/v6.0.2/esp-idf/export.sh"
idf.py -B "$HOME/.esp-idf-builds/focuscube-p4" \
  -p /dev/cu.usbmodemXXXX flash monitor
```

串口名称可能变化，不能长期写死。退出监视器按 `Ctrl+]`。

## 验收证据

- 屏幕固定中文照片。
- 状态区显示后端真实数据的照片或短视频。
- AI 复盘和提醒完整显示的照片或短视频。
- 串口日志中出现 Wi-Fi 获取 IP 和后端刷新成功记录。
- 临时断开后端后屏幕保留旧数据，恢复后自动刷新的短视频。
