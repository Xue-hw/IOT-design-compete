## 14:00 — 电源架构重构

## 16:30 — 代码修改汇总

### `config.h` — 完全重写
- 所有 GPIO 宏按原理图重新定义
- WiFi 宏：`WIFI_MODE_STA/AP` → `CFG_WIFI_MODE_SOFTAP/STA`
- MIC I2S：BCLK=42, WS=43, DIN=44（GPIO41 释放给传感器电源）

### `bsp_power.c` — 完全重写
- GPIO48 PWR_HOLD 推挽输出保持上电
- GPIO47 BOOT 按键输入上拉
- GPIO41 传感器电源域开关
- `bsp_power_safe_shutdown()`: WiFi off → SD unmount → GPIO41 LOW → GPIO48 LOW

### `bsp_led.c` — 重写为 RMT API
- 使用 `rmt_new_tx_channel()` + `rmt_new_bytes_encoder()`
- WS2816 时序：T0H=32 ticks, T1H=64 ticks (80MHz)

### `imu_task.c` — 完全重写
- QMA6100P → BMI270
- 寄存器：CHIP_ID=0x00, ACC_DATA_X_LSB=0x0C, GYR_DATA_X_LSB=0x12, CMD=0x7E
- 软复位 CMD=0xB6, accel ±4g, gyro ±500dps

### `web_control.c` — 修复
- snprintf buffer: full[256→512], url[256→512], row[512→1024], hdr[160→320]
- `/* ... */` 注释改为 `//`（避免 `*/` 导致的编译错误）

### `CMakeLists.txt` — 修复
- `esp_timer` 加入 REQUIRES
- `sys_events.c` 显式加入 SRCS

### `app_main.c` — 修复
- 移除已删除的 `bsp_power_woke_from_sleep()` 调用
