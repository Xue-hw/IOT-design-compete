# 物联网比赛项目资料总览

本目录是 FocusCube 当前比赛资料的统一入口。现在的主线已经定为：

**S3 立方体采集端 + ESP32-P4 七寸屏展示端 + 后端大模型 + Web/提交材料。**

## 当前作品定位

FocusCube 是一个面向学习/办公场景的智能光环境与专注管理系统：

```text
S3 立方体采集光照、姿态、触摸/按键、专注计时、电量等结构化数据
        ↓
后端接收并聚合数据
        ↓
云端大模型生成复盘、建议和提醒
        ↓
Web 看板与 P4 七寸屏展示结果
        ↓
S3 或 P4 显示/响应提醒
```

## 分工口径

| 成员 | 方向 | 核心目标 |
|---|---|---|
| A | S3 端 | 保持已烧录 S3 立方体稳定，负责采集、上传、提醒反馈 |
| B | P4 端 | 做出 P4 七寸屏展示节点，展示数据、大模型结果或系统状态 |
| C | 大模型 + 后端 | 接口、数据聚合、大模型复盘、提醒生成 |
| D | 前端 + 材料 | Web 看板、方案书、PPT、演示视频、提交打包 |
