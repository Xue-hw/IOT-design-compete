# FocusCube 当前硬件事实与 BOM

## 当前硬件口径

- S3 是已经烧录好的立方体采集端，后续不围绕屏幕尺寸和外壳尺寸做大改。
- P4 是七寸屏展示端，最终视频里应清楚展示 P4 屏幕。
- 采购和补件优先服务于“能演示闭环”，不要再扩展旧的录音/拍照/文件传输路线。

## S3 立方体采集端

| 模块 | 当前口径 | 说明 |
|---|---|---|
| 主控 | ESP32-S3 | 已作为采集端使用 |
| 形态 | 立方体 | 已烧录，尽量不大改 |
| 屏幕 | S3 自带/已有屏幕 | 可用于本地提示，不作为当前尺寸问题处理 |
| 传感数据 | 光照、姿态、触摸/按键、专注计时、电量 | 以现有固件实际可输出字段为准 |
| 提醒反馈 | 屏幕、LED、蜂鸣或状态提示 | 任选稳定方式 |

## P4 七寸屏展示端

| 模块 | 当前口径 | 说明 |
|---|---|---|
| 主控 | ESP32-P4 | 负责展示端 |
| 屏幕 | 7 寸屏 | 当前实际屏幕，必须按 7 寸 UI 设计 |
| 网络 | Wi-Fi/HTTP 优先 | 从后端拉取 status/report/reminders |
| 展示内容 | S3 状态、大模型复盘、提醒列表 | 优先保证大字号、清楚、稳定 |

## 后端与展示所需资源

| 项目 | 用途 |
|---|---|
| 后端服务器或本地电脑 | 接收 telemetry，调用大模型，提供接口 |
| 云端大模型 API Key | 生成中文复盘和提醒 |
| Web 看板运行环境 | 展示曲线、指标、报告 |

## 不再作为当前采购重点

- 给 S3 重新选屏幕或重新设计外壳。
- 为了旧路线采购新的录音、拍照、SD 扩展件。
- 购买不确定能在 7 月 27 日前接入闭环的复杂外设。
