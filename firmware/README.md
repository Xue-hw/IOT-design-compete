# 固件目录

| 目录 | 用途 | 当前状态 |
|---|---|---|
| [`esp32-c3/`](esp32-c3/) | AS7341 光照采集与电脑桥接 | 已上传并通过测试 |
| [`esp32-s3/`](esp32-s3/) | FocusCube 立方体采集端 | 等待可用固件 |
| [`esp32-p4/`](esp32-p4/) | P4 七寸屏展示端 | 等待成员 B 上传 |
| [`arduino/`](arduino/) | 历史 Arduino 智能家居原型 | 已归档 |

`esp32-c3/` 是 S3 到位前的真实光照临时采集链路。最终比赛主线仍使用 `esp32-s3/` 和 `esp32-p4/`。Arduino 目录用于保存早期原型，不应在方案书中描述为 FocusCube 当前固件。
